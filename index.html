🔥 Conectar Firebase
InchPick necesita Firebase Realtime Database para sincronizar todas las tablets. Es 100% gratuito.
1
Crear proyecto Firebase
Ve a console.firebase.google.com → nombre: inchpick-cd → Crear.
2
Activar Realtime Database
Compilación → Realtime Database → Crear base de datos → us-central1 → Modo de prueba → Habilitar.
3
Copiar firebaseConfig
Engranaje ⚙ → Configuración → Tus apps → Web → copia el bloque firebaseConfig.
const firebaseConfig = { apiKey: "AIzaSy...", databaseURL: "https://inchpick-cd-default-rtdb.firebaseio.com", projectId: "inchpick-cd", ... };
4
Pega tu configuración

Continuar sin Firebase (modo demo local)
